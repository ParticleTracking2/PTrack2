Descripción detallada

Clase para realizar convolucion de imagen mediante FFTW. Esta clase ocupa FFTW probado y funcionando en version 3.3. Además para evitar reserva y liberacion de memoria constantemente, usa Cache de datos. Para evitar calculos redundantes, estos se calculan una sola vez y se almacenan para posterior uso. En su primer uso se ejecutan 3 Threads de FFTW y luego solo se ejecutan 2 pues un resultado es constante.
