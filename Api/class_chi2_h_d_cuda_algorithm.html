vector< MyPeak > Chi2HDCudaAlgorithm::run ( ParameterContainer *  pc) [virtual]

Ejecuta el algoritmo de minimos cuadrados para detectar Peaks usando GPU. Esta variante ejecuta en forma iterativa la deteccion de minimos cuadrados en la diferencia de la imagen contra la imagen generada a partir de los peaks encontrados, produciendo una mayor deteccion de particulas.

Parámetros:
pcParametros que ocupa el algoritmo para correr.
Devuelve:
Vector de MyPeak representando los peaks encontrados dentro de los datos entregados.
